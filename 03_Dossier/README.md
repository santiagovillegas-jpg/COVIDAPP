# Dossier de Documentación Técnica - COVIDAPP

Este espacio centraliza toda la documentación técnica consolidada y los informes de ingeniería detallados que respaldan el desarrollo del proyecto **COVIDAPP**. La documentación sigue estándares internacionales y metodologías académicas de ingeniería.

## Contenido Técnico Principal
- **CDIO III (`cdioIII.pdf`):** Documento maestro detallando las etapas de Concebir, Diseñar, Implementar y Operar del sistema integral para la institución **Covida**.
- **Informes Técnicos:** Análisis profundos sobre la arquitectura del software, selección de tecnologías y evaluaciones de desempeño.
- **Manuales de Operación:** Guías técnicas para administradores sobre la gestión y el mantenimiento de la plataforma.

## Metodología de Documentación
1.  **Enfoque de Ingeniería:** Basado en el marco de trabajo CDIO para asegurar la integralidad técnica.
2.  **Calidad Documental:** Estándares de redacción formal y técnica para garantizar la claridad y utilidad de la información.
3.  **Actualización Continua:** La documentación se actualiza en paralelo con cada nueva iteración del software.

# 📚 03_Dossier: Documentación y Evidencias

Este directorio compila toda la documentación técnica formal, manuales y evidencias visuales del proyecto **COVIDAPP**. Es el repositorio central para la justificación técnica y funcional de la solución.

## 📁 Contenido Destacado

*   **📜 [cdioIII.pdf](./cdioIII.pdf):** El documento maestro bajo el estándar **CDIO (Concebir, Diseñar, Implementar y Operar)**. Detalla la ingeniería, arquitectura y decisiones técnicas del proyecto.
*   **📋 [Matriz Objetivos y Metas.xlsx](./Matriz%20Objetivos%20y%20Metas.xlsx):** Seguimiento del cumplimiento de los requisitos iniciales.
*   **🖼️ Evidencias de Diseño (Screenshots):** Capturas de pantalla que demuestran la interfaz de usuario para diferentes roles:
    *   `Login_Admin.png`, `Login_User.png`: Interfaces de acceso.
    *   `Médicos.png`, `Pacientes.png`: Vistas específicas por rol.
    *   `Administrativos no médicos.png`: Panel de control institucional.

## 🛠️ Manuales y Guías

*   **[Plantilla_Manuales.pdf](./Plantilla_Manuales.pdf):** Estructura base para los manuales de usuario y técnicos.
*   **[Terminado(DoD).xlsx](./Terminado(DoD).xlsx):** Definición de "Hecho" (Definition of Done) para asegurar la calidad de cada entrega.

## 🚩 Propósito del Proyecto
**COVIDAPP** busca erradicar la ineficiencia administrativa en las instituciones de salud mediante:
1.  **Agendamiento Inteligente:** Reducción del ausentismo.
2.  **Gestión Centralizada:** Datos accesibles y seguros.
3.  **Interfaces Intuitivas:** Enfoque en el usuario final.
