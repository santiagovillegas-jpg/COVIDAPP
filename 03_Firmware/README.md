# Desarrollo de Firmware (COVIDAPP)

Esta sección contiene el código fuente desarrollado para el control del hardware asociado al proyecto COVIDAPP.

## Estructura del Código
- **`main.c`**: El punto de entrada principal que inicializa el sistema y ejecuta el ciclo de control.
- **`driver/`**: (Próximamente) Módulos de bajo nivel para periféricos (ADC, UART, I2C).
- **`logic/`**: (Próximamente) Lógica de procesamiento y algoritmos de captura de datos.

## Herramientas de Desarrollo
* **Lenguaje:** C/C++
* **IDE Recomendado:** VS Code con extensiones de PlatformIO / STM32CubeIDE (según el hardware).
* **Compilador:** GCC-based toolchains.

## Instrucciones de Compilación
Actualmente el firmware se encuentra en fase de desarrollo inicial (scaffolding). Las instrucciones de compilación específicas serán añadidas una vez se estabilice el núcleo del sistema.
