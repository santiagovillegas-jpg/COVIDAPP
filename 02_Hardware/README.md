# Diseño de Hardware (COVIDAPP)

En este directorio se encuentra el diseño físico y la arquitectura de hardware necesaria para el funcionamiento del dispositivo complementario a la plataforma web.

## Componentes Principales
* **Microcontrolador:** Encargado del procesamiento de datos y control de periféricos.
* **Módulos de Comunicación:** Para la transmisión de datos a la base de datos central.
* **Sensores:** Para la captura de información relevante del paciente/usuario.

## Contenido del Directorio
1. **Esquemáticos:** Diagramas de conexión del circuito.
2. **PCB Layout:** Diseño físico de la placa de circuito impreso (en desarrollo).
3. **Datasheets:** Especificaciones técnicas de los componentes seleccionados.

## Especificaciones Técnicas
Las especificaciones detalladas se encuentran en el archivo [BOM](../01_Gestion/README.md) del proyecto.
